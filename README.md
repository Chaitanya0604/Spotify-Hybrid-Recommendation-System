# 🎵 Spotify Hybrid Recommendation System

A personalized music recommendation system that combines **Content-Based Filtering** and **Collaborative Filtering** techniques to recommend relevant songs to users.

🚧 **Current Progress:** Content-Based Filtering module completed. Collaborative Filtering and Hybrid Recommendation Engine are under development.

## Project Overview

The goal of this project is to build a scalable Spotify recommendation system capable of generating personalized song recommendations using both song metadata and user listening behavior.

The final system will combine:

* Content-Based Filtering
* Collaborative Filtering
* Hybrid Recommendation Engine

## Completed Module: Content-Based Filtering

The content-based recommender suggests songs similar to a selected song using Spotify metadata and audio features.

### Features Used

* Artist
* Release Year
* Tags
* Danceability
* Energy
* Loudness
* Speechiness
* Acousticness
* Instrumentalness
* Liveness
* Valence
* Tempo
* Key
* Time Signature

### Feature Engineering

| Feature Type                | Technique            |
| --------------------------- | -------------------- |
| Numerical Features          | MinMaxScaler         |
| Artist, Key, Time Signature | OneHotEncoder        |
| Release Year                | CountEncoder         |
| Tags                        | TF-IDF Vectorization |

A `ColumnTransformer` was used to combine all preprocessing steps into a single pipeline.

### Similarity Computation

Songs are transformed into numerical vectors and stored as a sparse feature matrix.

Cosine Similarity is used to identify songs with similar characteristics:

```python
cosine_similarity(song_vector, transformed_data)
```

Recommendations are generated by selecting the songs with the highest similarity scores.
